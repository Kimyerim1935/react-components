# Base Components

## Components

- Backdrop (Overlay)
- Click Away Listener
- Divider
- Form Control
- Listbox

### Backdrop

- Focus Trap
- Dimmed Overlay / Transparent Overlay
- Overflow Hidden
- Combobox에서 사용 (Autocomplete, Select)

### Click Away Listener

- Global Mouse Event Listener
- Popover에서 사용

### FormControl

- Label Text & Helper Text
- Select, TextField 등에서 사용
